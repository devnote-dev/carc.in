# [carc.in](http://carc.in) - compile and run code in

A web service that runs your code and displays the result.

Consists of:

* Backend API server
* Frontend
* Sandbox manager (designed for Archlinux host with an BTRFS filesystem available)

## Supported languages

* Crystal
